Latest Changes

Recent updates to the DigiLawyer Design System.

v2.8.0
MINOR
Scaffold tooling — tokens.json export + component generator
25 May 2026
Two new scripts in docs/scripts/. build-tokens.js exports tokens.json (323 light + 323 dark, var() refs resolved) so React, AI tools, and Figma plugins can consume tokens. new-component.js scaffolds a component page and prints paste-blocks for the manual edits.
v2.7.0
MINOR
Radio Button component — full documentation
22 May 2026
New .radio component implemented 1:1 from Base-AI Figma. Same a11y pattern as Checkbox; browser handles single-select via shared name. 13 new --radio-* tokens, matrix is 2×3 matching Figma. Search index extended.
v2.6.0
MINOR
Persistent global header — logo, version chip, search, theme toggle
22 May 2026
New sticky top bar auto-injected by ds.js on every page. Global search across pages, sections, and ~120 semantic CSS tokens with keyboard nav. Sidebar lost its own logo/theme button — moved to the header.
v2.5.0
MINOR
Checkbox component — full documentation
22 May 2026
New .checkbox built on native input — 3 variants × 3 states, implemented 1:1 from Figma. 13 new --checkbox-* tokens (light + dark), two new sprite symbols (#ico-check, #ico-dash). 5-section docs with 3×3 matrix mirroring Figma.
v2.4.0
MINOR
Button Group component — full documentation
22 May 2026
.btn-group composes existing button variants. Attached (segmented control) and separated styles, vertical orientation, toggle/selection via aria-pressed. Documented at components/button-group.html with 9 sections, interactive demos, and an API table.
v2.3.0
MINOR
Audit pass — auto-TOC, bulk migration, diagnostics
22 May 2026
ds.js auto-extracts the TOC from h2[id] anchors. Bulk-migrated all 26 component stubs to the lean v2.3 shell (~25 lines each). Added DS_PAGE diagnostics and a ?ds-debug toggle. Starter templates for new pages.
v2.2.0
MINOR
Shared TOC component, head-loaded ds.js
22 May 2026
ds.js now renders the on-page TOC from a window.DS_PAGE.toc array, eliminating duplicated scroll-spy code. Moved ds.js to <head> so theme restore fires before first paint (no flash). Migrated all 10 built pages.
