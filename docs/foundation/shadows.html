Shadows

Five elevation levels — subtle (1) to strong (5). Light mode uses black alpha, dark mode uses white alpha. Always use --shadow-* tokens, never hardcode box-shadow values.

Levels

Each level increases both spread and alpha, creating a stronger sense of elevation.

LevelTokenElevation
1--shadow-1Subtle — cards at rest, inline panels
2--shadow-2Resting cards with slight lift
3--shadow-3Dropdowns, floating toolbars
4--shadow-4Modals, drawers
5--shadow-5Strongest — overlays, command palette

Light mode

Black alpha shadows on white/light surfaces.
